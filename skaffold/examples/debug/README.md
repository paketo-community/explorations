# Getting Started

## Installing Skaffold

1. Download and install the `skaffold` CLI:
   ```
   brew install skaffold
   ```
1. Download and install `minikube`:
   ```
   brew install minikube
   ```
1. Start the `minikube` VM:
   ```
   minikube start
   ```

## Running the example

1. Run `skaffold debug`
